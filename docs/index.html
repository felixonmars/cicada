(jojo

  (* jojo is a language
     employs an implementing technic of Forth
     called threaded code interpreter)

  (* jojo is written in portable C
     to be embedded in C and C++)

  (* jojo has a low level untyped core
     can be used like Forth)

  (* jojo's core provides :

     (* a mechanism for dynamic typing)

     (* an object system like smalltalk)

     (* a module system like node.js)

     (* a low level syntax extending mechanism,
        in which syntactic keywords are threaded code compilers.
        it can be used as flexible as lisp macro)))
